# Job Application Tracking Dataset

A simulated dataset of 30 job applications submitted during a
semester-long job search, containing application status, salary,
location, cost of living, and source information.

## Usage

``` r
jobs
```

## Format

A data frame with 30 rows and 8 variables:

- company:

  Character. Name of the company applied to.

- position:

  Character. Job title of the applied position.

- location:

  Character. City and state of the job location.

- cost_of_living:

  Numeric. Cost of living index where 100 equals the US average.

- status:

  Character. Current application status: Applied, Interviewed, Offered,
  or Rejected.

- date_applied:

  Date. The date the application was submitted.

- salary:

  Numeric. Advertised or estimated annual salary in USD.

- source:

  Character. Platform used to find the job: LinkedIn, Indeed, Referral,
  or Career Fair.

## Source

Simulated data created for demonstration purposes.
